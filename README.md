# 🎮 Quiz Game in C

A simple **terminal-based Quiz Game** written in the **C programming language**.
This project allows users to test their knowledge by answering multiple-choice questions and receiving a score at the end.

The program is **menu-driven**, interactive, and beginner-friendly. It demonstrates core programming concepts such as **switch-case statements, conditional logic, loops, and user input handling**.

---

## 📌 Features

✨ Interactive terminal-based quiz
✨ Menu-driven program
✨ Multiple-choice questions
✨ Score calculation system
✨ Instant feedback for answers
✨ Final performance result

## 🎯 Game Rules

* The quiz contains **5 questions**.
* Each question has **4 options (A, B, C, D)**.
* Each correct answer gives **1 point**.
* There is **no negative marking**.
* Your **final score** will be displayed at the end.
